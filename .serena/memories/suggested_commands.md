# Suggested Commands
- Install deps: `pnpm install`
- Dev server (Vite): `pnpm dev`
- Type check + build: `pnpm build`
- Preview build: `pnpm preview`
- Biome/Ultracite check: `pnpm check`
- Auto-fix formatting/lint: `pnpm fix`
- Install git hooks: `pnpm prepare` (runs `lefthook install`)
- Run audit (pre-push hook): `pnpm audit`
